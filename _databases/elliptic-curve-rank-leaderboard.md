Inspired by Dujella's rank tables, this site tracks elliptic curves over Q of high Mordell-Weil rank relative to their size (log conductor, naive height, Faltings height, or log discriminant), displayed as interactive rank-vs-size scatter plots and a browsable table. Users log in with GitHub to submit curves with Weierstrass coefficients and rational points, whose independence is certified via Neron-Tate height computations, and the full database is downloadable as JSON.
