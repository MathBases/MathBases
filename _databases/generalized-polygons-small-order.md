A list of known generalised n-gons of small order for n = 4, 6, 8 (s,t > 1): all known generalised quadrangles with st < 100, generalised hexagons with st < 40, and generalised octagons with st < 20, giving for each a text file of point-line incidences and a file of automorphism group generators, plus elementary divisors, automorphism group orders and orbit lengths.
