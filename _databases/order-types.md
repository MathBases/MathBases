Complete database of all order types of n points in the plane in general position, realized by integer-coordinate point sets, for 3 <= n <= 11, together with companion files of extremal values per order type (rectilinear crossing number, numbers of triangulations, crossing-free Hamiltonian cycles, spanning trees and matchings, k-gons and empty k-gons, k-sets, convex cover/partition/decomposition numbers, extreme points). Files for up to 10 points are downloadable; the 11-point extension (2,334,512,907 order types, 96 GB) is available on request.
