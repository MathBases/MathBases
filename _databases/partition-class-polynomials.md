Text files giving the partition class polynomials H_n(X) for positive integers n up to 750, as defined in the Bruinier-Ono-Sutherland paper 'Class polynomials for nonholomorphic modular functions'. Each H_n is a monic polynomial over Q of degree equal to the class number h(1-24n), whose trace is (24n-1)p(n), where p(n) is the nth partition number — making the finite algebraic formula for p(n) explicit.
